# 3D Solar System

A 3D interactive solar system built with HTML, CSS, and JavaScript. This project visually represents the solar system, allowing users to see the planets, their orbits, and more through a simple web interface.

## Demo

You can view the demo on my [YouTube channel](https://www.youtube.com/@WajahatWebDeveloper).

## Features

- Interactive 3D visualization of the solar system
- Clickable planets to display basic information
- Control panel for adjusting view settings like speed, size, and distance
- Realistic orbit animations

## Built With

- **HTML**: For the page structure
- **CSS**: For styling and animations
- **JavaScript (jQuery)**: For dynamic behavior and interactivity
